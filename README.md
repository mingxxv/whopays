# WhoPaysClaude

A Windows 95-style app to track who last paid for Claude Code. Built with Next.js.

## Features

- Windows 95 themed UI
- Tracks the last person who paid for Claude Code
- State stored in the backend for persistence
- Deployable to Vercel

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This app is configured for deployment on Vercel. Simply connect your repository to Vercel for automatic deployments.

## How It Works

The app maintains a simple state in the backend that stores who last paid for Claude Code. Click either the "Gordon" or "Zhuang" button to update the payer.
